Minecraft is a 3D sandbox game, made in 2011. In the game, there is a prodecurally generated world that you can gather resources, build strutures, make items and beat the Ender dragon.

The game has single player, and a multiplayer gamemode. The game can be heavily modded, as you can access the code and use mods from the internet.

Anyone can host a multiplayer game, from their own computer or using a server host. On multiplayer servers, you can use plugins to code and make the server into anything you want!

The game was originally developed in Java, but since coming out has a C++ edition for mobile players.
